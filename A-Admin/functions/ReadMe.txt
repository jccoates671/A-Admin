Hi There And Thank You For Installing A Admin. If You Cant Guess. A Admin Is A Server Administration Function Pack
A Admin Has Many Commands, But First You Need To Know About How To Use Them.
All A Admin Commands For Adminstrators Have An "A-" Before The Command
All Player Commands Have No Prefix
A Admin Has 4 Command Groups
Owner
Admin
Mod
User
Each Group Granting Different Access
Commands Can Be Found In The Commands Folder
To Grant A User A Group Simply Type In Game "tag <player name> add <group name>"
an example is
tag Bot19 add Mod
You Also Can Revoke Privliges By Typing
tag Bot19 remove Mod
